## Run Locally

Clone the project

```bash
  git clone https://github.com/ishubhamrathi/UMS/
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install localStoragePy==0.2.3
  pip install streamlit==1.16.0
  pip install  ums==0.3.0
```

Start the server

```bash
  streamlit run login.py
```
